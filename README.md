# Real-Time Chess Contest App

## Setup

### Backend

1. cd backend
2. npm install
3. npm run dev

### Frontend

1. cd frontend
2. npm install
3. npm start

## Features implemented
- JWT auth
- wallet top-up + history
- room create/join
- socket matchmaking + real-time moves
- chess logic using chess.js
- prize distribution service
- admin user/room/transaction view APIs
